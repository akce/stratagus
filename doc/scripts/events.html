ResolutionChanged

Intro - Introduction to stratagus events

Stratagus Events allow for games to react to changes in the Stratagus engine.

Functions

ResolutionChanged()

Video resolution has been changed.

This event will be generated only when DynamicResolution is turned on and the containing Window has been resized.

Games can check Video.Width and Video.Height or call GetVideoResolution to find the new video resolution.

Example

    function ResolutionChanged()
        new_width, new_height = GetVideoResolution()
        ReCalculateGameScreen(new_width, new_height)
        RedrawGameScreen()
        ..
        ..
    end
